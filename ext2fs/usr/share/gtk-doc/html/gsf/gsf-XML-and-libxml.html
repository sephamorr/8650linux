enum GsfXMLContent

typedef enum {
	GSF_XML_NO_CONTENT = FALSE,
	GSF_XML_CONTENT,
	GSF_XML_SHARED_CONTENT
} GsfXMLContent;

Controls the handling of character data within a parser node.

GSF_XML_NO_CONTENT

 node has no cstr contents

GSF_XML_CONTENT

 node has cstr contents

GSF_XML_SHARED_CONTENT

 node has contents that is shared with children
